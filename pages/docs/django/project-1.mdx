## Quick Start

Fork the repository and then clone it locally by doing -

```bash copy
git clone https://github.com/dev-mdirfan/simple-crud-django-app.git
```

**Install Dependencies:**

```bash copy
pip install -r requirements.txt
```

**Or:**

Copy and paste the below command in your terminal -

```bash copy
pip install django
```

**Run Server:**

```bash copy
python manage.py runserver
```
