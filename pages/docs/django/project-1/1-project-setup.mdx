# 1. Project Setup

If you are new to Django. Please go through the [poject setup](/django) steps to setup your environment. Otherwise you can continue.

## Create a new project


```bash copy
django-admin startproject mysite
```

Your project structure should look like this:

<FileTree>
    <FileTree.Folder name="mysite" defaultOpen>
        <FileTree.Folder name="mysite" defaultOpen>
            <FileTree.File name="__init__.py" />
            <FileTree.File name="asgi.py" />
            <FileTree.File name="settings.py" />
            <FileTree.File name="urls.py" />
            <FileTree.File name="wsgi.py" />
        </FileTree.Folder>
        <FileTree.File name="manage.py" />
    </FileTree.Folder>
</FileTree>
